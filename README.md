# Customer-Churn-Prediction

# Problem Statement
Customer churn is a major issue for subscription-based businesses. The goal is to predict whether a customer will leave the company so that retention strategies can be implemented.

# Objectives
Analyze customer behavior
Identify key churn drivers
Build a predictive model
Provide actionable business insights

# Dataset
Telco Customer Churn Dataset (Kaggle)

# Technologies Used
Python
Pandas, NumPy
Seaborn, Matplotlib
Scikit-learn

# Workflow
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Model Building (Random Forest, Logistic Regression)
Model Evaluation (Recall, ROC-AUC)
Feature Importance Analysis

# Key Insights
Customers with month-to-month contracts are more likely to churn
Higher monthly charges increase churn probability
Customers with short tenure are at higher risk

# Model Performance
Achieved strong performance using Random Forest
Focused on Recall to capture maximum churn cases

# Business Impact
Helps identify high-risk customers
Enables targeted retention strategies
Reduces revenue loss
